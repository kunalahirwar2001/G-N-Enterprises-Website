# GN Enterprises Website

Static responsive website for GN Enterprises, a solar panel expert company in Gwalior, Madhya Pradesh.

## Structure

- `index.html` - main website
- `css/style.css` - styling and responsive glassmorphism UI
- `js/script.js` - vanilla JavaScript interactions
- `assets/logo/` - company logo
- `assets/site-photos/` - project/site photos for gallery and hero

## Updating Photos

Replace files in `assets/site-photos/` while keeping the same names:

- `site-photo-01.jpg`
- `site-photo-02.jpg`
- `site-photo-03.jpg`
- `site-photo-04.jpg`
- `site-photo-05.jpg`
- `site-photo-06.jpg`
- `site-photo-07.jpg`
- `site-photo-08.jpg`

The website will automatically use the updated images.
